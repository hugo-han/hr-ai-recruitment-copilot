# OPC AI Software Factory
# Enterprise Rules


# 企业级研发规范


## AI使用原则


AI是研发协作者，不替代工程治理。


所有AI输出必须：

- 可解释
- 可审计
- 可追踪



---

# 安全要求


禁止：

- 泄露敏感信息
- 提交账号密码
- 提交Token
- 修改生产配置



---

# 代码质量


所有代码：

必须：

- 有明确目的
- 有必要测试
- 有Review记录



---

# 文档要求


设计先行。


禁止：

没有设计文档直接开发。



---

# 变更管理


所有生产变更必须：

Issue

↓

Review

↓

Release



---

# AI Agent权限


AI Agent：

可以：

- 创建代码
- 创建文档
- 提供建议


需要人工确认：

- 架构调整
- 数据库变更
- 生产发布
